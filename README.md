# BankingApp
MVP for Banking App

1. Clean and build all the microservices using maven commands.
2. Execute the docker compose commands to build and run the services in the local docker environment setup.
docker compose down -v
docker compose up --build -d
command to validate logs for individual service - eg. - docker logs config-service
4. Validate if services are up and running.  
5. Use postman to verify the backend rest services.


Verification steps - 
1) Service registry - http://localhost:8761/
2) postman collection - Available in the project folder
